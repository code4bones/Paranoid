Для установки просто запустить программу из той директории, где она в дальнейшем будет жить,
этот путь будет прописан в автозапуск.

Параметры:
 Общие:
  --silent       -Без диалогов во время установки или удаления
  --show         -Если программа уже работает, то запуск с этим ключем откроет диалог параметров работающего экземпляра
 Установка (не обязательные):
    не требуется, просто запусить ( или с --silent )	
 Удаление:
  --remove       -Программа будет удалена из автозапуска и работающий экземпляр программы закрыт
